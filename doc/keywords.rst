Keywords
========

* ``fn`` declare a function
* ``ret`` return a value from a function
* ``use`` include/import another module
* ``if``
* ``elif``
* ``else``
* ``while``
* ``for``

Conditions example::

	if (something) {

	} elif (something else) {

	} else {

	}

	bit operators: and, or, not
	null equals to false in conditions

Loop examples::

	n: i32 = 5
	for (i: i32 = 0; i < 5; i++)
		print(i)

	output: 12345

	while (statement) {
		...
	}
